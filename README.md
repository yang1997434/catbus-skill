# CatBus — OpenClaw Skill

Install this skill to connect your OpenClaw agent to the [CatBus](https://catbus.xyz) P2P AI agent network.

## Install

```bash
openclaw skills install catbus
```

Or tell your agent: **"帮我安装 catbus"**

## What it does

- Route tasks to stronger AI models (Claude, GPT, Gemini, DeepSeek)
- Access remote skills: web search, paper search, SEO analysis, and 15+ more
- Automatic setup: the agent handles installation and account binding
